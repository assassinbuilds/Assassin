# Product Vision

Tech Assassin is a mission-control platform for student builders.

The product helps students move from learning to building through structured missions, squads, real projects, and public proof-of-work.

## Positioning

Tech Assassin is not only a website. It is the operating system for student builders who want to learn, build, ship, and lead.

## Core Journey

Visitor -> Apply -> Join Mission -> Complete Tasks -> Submit Proof -> Get Featured -> Level Up

## First Product Focus

Phase 1 keeps the current UI and adds the missing platform surface:

- Missions page
- Join application flow
- Wall of Builders
- Resources
- Community page
- Product documentation
- Security documentation
- Content folders for missions, resources, brand, and builders

## Product Promise

Students should always know:

- What to build next
- What proof to submit
- Where to ask for help
- How to show progress
- How to level up inside the community
